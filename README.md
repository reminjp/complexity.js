# complexity

[![License][license-badge]][license-badge-url]

A static code analyzer written in JavaScript.

## Usage

```bash
yarn add @remin/complexity
```

## Development

- `yarn`
- `java` (for ANTLR4)

```bash
# install dependencies
$ yarn install

# build for production
$ yarn run build

# test the built package
$ yarn run test
```
